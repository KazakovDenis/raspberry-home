# Raspberry Pi Home
Домашний проект на VPS + Raspberry Pi 4 Model B.

## Компоненты
- Панель управления
- Видеонаблюдение
- Сетевое файловое хранилище
- Бэкап хранилища в облако

## Требования

### Панель управления
- Доступ к функциям всех компонентов
- Настройка компонентов
- Запуск / остановка компонентов
- Мониторинг ресурсов

### Видеонаблюдение
- Просмотр видео онлайн
- Запись видео в файл:
  - включение записи
  - остановка записи
- Просмотр фото
- Сохранение фото
