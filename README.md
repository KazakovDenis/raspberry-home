# Raspberry Pi Home
Pet project using VPS + Raspberry Pi.

## Components
| Component    | Tech                      | 
|--------------|---------------------------|
| Control unit | Telegram bot / Python     |
| VPN          | Wireguard                 |
| DNS          | Ubound                    |
| Adblock      | Pihole                    |
| CCTV         | Motion + Arecord + FFMpeg |
| NAS          | Smb + HDD                 |
| Backup       | Rclone + Yandex.Disk      |
| Monitoring   | Grafana, Prometheus       |
