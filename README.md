# Raspberry Pi Home
Домашний проект на Raspberry Pi 4 Model B.

## Компоненты
- Панель управления
- Видеонаблюдение
- Сетевое файловое хранилище

## Требования

### Панель управления
- Доступ к функциям всех компонентов
- Настройка компонентов
- Запуск / остановка компонентов

### Видеонаблюдение
- Просмотр видео онлайн
- Запись видео в файл:
  - включение записи
  - остановка записи
- Просмотр фото
- Сохранение фото
